# Mathematical Induction

## 1-form Mathematical Induction

- $P(0)$ True.
- $P(n + 1)$ True if $P(n)$ True.
  
Then $\forall n \in N$ we have $P(n)$ True. 



## 2-form Mathematical Induction

- $P(1), P(2), \ldots, P(k)$ True.
- $P(1..k)$ True implies $P(k + 1)$ True.

Then $\forall n \in N$ we have $P(n)$ True.
